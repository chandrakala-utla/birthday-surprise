# 🎂 Birthday Surprise Website ❤️

A beautiful and interactive birthday surprise website created with **HTML, CSS, and JavaScript**. The website takes the user through a magical journey filled with animations, memories, music, and a heartfelt letter.

## 🌐 Live Demo

🔗 https://chandrakala-utla.github.io/birthday-surprise/

---

## ✨ Features

- 🔒 Password Protected Entry
- 🎁 Interactive Gift Opening Animation
- ✨ Animated Loading Screen
- 📸 Beautiful Memory Gallery
- 🎂 Birthday Cake with Blow Candle Animation
- 💌 Premium Envelope Opening Effect
- ❤️ Heartfelt Letter with Typewriter Animation
- 🎵 Background Music for the Letter
- 🌙 Animated Night Sky
- ⭐ Twinkling Stars
- 💖 Floating Hearts
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/chandrakala-utla/birthday-surprise.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

Or simply visit the live website:

👉 https://chandrakala-utla.github.io/birthday-surprise/
